## Reguli
- Tranziții hover: 150–200ms ease-out (culoare + umbră roz `0 4px 14px rgba(255,45,120,.35)`).
- Radius: butoane 999px (pill), carduri 18px, badge-uri mari.
- Gradient CTA: `linear-gradient(120deg, #FF2D78, #E01B63)`.
- Text alb pe roz = OK butoane. Text roz pe alb = doar bold / mărimi mari.
- Compat badge culori: verde >80%, galben 50–80%, gri <50%.
